#### 像素响应时间

刷新率和响应时间没有必然的联系。但像素响应时间应该尽力做到低于一个刷新周期。  
当响应时间接近甚至超出刷新周期时，会引入一些动态模糊，拖影或者颜色错误的问题。

### 应用：更改高刷

### 高刷给工程师带来哪些挑战？

1. 屏幕自身素质表现，像素响应时间/屏幕拖影/像素阻力，过程会导致拖影
2. SOC能力：GPU要渲染，CPU要计算，带宽能力

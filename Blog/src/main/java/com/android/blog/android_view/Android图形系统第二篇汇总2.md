## 1、什么是屏幕刷新率

刷新率的概念诞生自CRT显示器，指的是单位时间内屏幕刷新的次数，表示单位为HZ

60HZ屏幕就是显示器一秒钟显示60次画面，即1/60秒内出一个画面

CRT显示器是使用“逐行扫描”技术来刷新页面的，

### 什么情况下不建议开启垂直同步？

1. 游戏玩家，对帧数延迟等敏感
2. CPU/GPU垃圾的设备，屏幕本身60HZ，GPU每秒最多50帧，开启垂直同步后，显卡还得等屏幕处理完再去绘制下一帧，于是乎就更慢了
